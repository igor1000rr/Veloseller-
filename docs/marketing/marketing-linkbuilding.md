## 1. Что продаём (позиционирование)

**Veloseller — сервис аналитики остатков и продаж для селлеров Wildberries и Ozon.**
Подключаете кабинет по API — видите единую картину по всем складам (FBO/FBS) и получаете деньги, которые сейчас теряются.

**Ключевые ценности → боли клиента:**

| Боль селлера | Что даёт Veloseller |
|---|---|
| Уходит в out-of-stock → теряет выручку и позиции | Контроль остатков по всем складам + алерты в Telegram |
| «Замороженные» деньги в неликвиде | Выявление мёртвых остатков (dead inventory) |
| Нет единой картины FBO/FBS, учёт в Excel руками | Авто-синк WB/Ozon + дашборды и авто-отчёты (Excel/PDF) |
| Не понимает юнит-экономику (комиссии, потерянная выручка) | Расчёт юнит-экономики, потерянной выручки, эластичности цены |
| «Что закупать / что продавать» | **Radar** — поиск ниш и новинок через Wordstat по брендам |

**УТП (1 фраза):** *«Veloseller показывает, сколько вы теряете на стокаутах и неликвиде — и помогает это вернуть, без ручного Excel».*

## 2. Кому продаём (ICP)

1. **Растущие селлеры WB/Ozon**, 100–5 000 SKU, кто уже теряет на стокаутах/неликвиде и ведёт учёт вручную.
2. **Менеджеры маркетплейсов и агентства**, ведущие несколько кабинетов (нужна одна панель на всех).
3. **Начинающие селлеры** с амбициями роста (Radar — «что продавать»).

Где они «обитают»: Telegram-чаты и каналы про WB/Ozon, VK-сообщества селлеров, YouTube/Telegram-блогеры по маркетплейсам, профильные курсы и конференции, поиск Яндекса по запросам про учёт/аналитику.

## 3. Цели и воронка (KPI)

**Воронка:** Узнаваемость → Трафик → Регистрация (trial) → **Активация** (подключил склад и увидел первый инсайт) → Оплата → Удержание/Реферал.

| Этап | Метрика | Ориентир на старте |
|---|---|---|
| Трафик | визиты/мес (орг + платн.) | рост x2 за квартал |
| Регистрация | trial sign-ups/мес | базовая цель — задать самим |
| Активация | % подключивших API за 24 ч | ≥ 50% |
| Оплата | trial → paid | 15–25% |
| Экономика | CAC < LTV (минимум x3) | контролировать с 1-го дня |
| Удержание | месячный churn | < 5–7% |
| SEO/ссылки | реф-домены, ИКС, позиции по кластерам, реф-трафик | растущая динамика |

> ⚠️ **Про соц.доказательство:** сейчас платящих единицы — публично числами клиентов не давить. Вместо этого: 2–3 **кейса** с цифрами «вернули X₽ выручки / разморозили Y₽ в неликвиде» от текущих платящих + «обрабатываем 14 000+ SKU». Кейсы — топливо и для лендинга, и для линкбилдинга.

## 4. Каналы привлечения (приоритеты для ранней стадии)

**Приоритет 1 — быстрые и дешёвые:**
1. **Сообщества селлеров** (Telegram/VK-чаты и каналы по WB/Ozon). Заходим экспертизой: разборы, ответы, полезные расчёты — **не спам**. Самый быстрый источник первых клиентов.
2. **Контент + SEO** — фундамент органики (см. §6). Линкбилдинг работает только поверх хорошего контента.
3. **Инфлюенсеры по маркетплейсам** (Telegram/YouTube) — интеграции, на старте часто по **бартеру** (доступ + комиссия).

**Приоритет 2 — среднесрок:**
4. **Линкбилдинг** — рост видимости и траста (детально в §5).
5. **Партнёрства/интеграции**: фулфилмент, бухгалтерия для селлеров, репрайсеры, курсы/школы маркетплейсов — кросс-промо и взаимные ссылки.
6. **Реферальная и партнёрская программа** (агентствам — % с оплат).

**Приоритет 3 — платный трафик (после первых сигналов PMF):**
7. **Яндекс Директ** на горячие коммерч. ключи; **Telegram Ads** и **VK Ads** на аудитории селлеров.

## 5. ЛИНКБИЛДИНГ — детальный план (главный запрос)

### 5.1. Зачем нам ссылки
- Рост **траста молодого домена** и видимости в Яндексе/Google.
- **Реферальный трафик** с тематических площадок и сообществ.
- **Бренд-сигналы** (упоминания Veloseller, даже без ссылки — Яндекс их учитывает).

### 5.2. Подготовка (до закупки — обязательно)
1. **Яндекс.Вебмастер** и **Google Search Console** — подтвердить домен, загрузить sitemap, проверить индексацию. *(Это можно сделать прямо сейчас — у меня есть доступ к этим инструментам, помогу технически.)*
2. **Снять текущий ссылочный профиль** (Я.Вебмастер → «Внешние ссылки», + ahrefs / keys.so / Megaindex) — точка отсчёта и список анкоров.
3. **Семантика и кластеры** (см. §6) — под них формируем анкоры и страницы-акцепторы.
4. **Базовая «брендовая» регистрация** (даёт первые естественные ссылки и сигналы):
   - Яндекс.Бизнес, 2ГИС, справочники.
   - Соцсети/каналы бренда (VK, Telegram, Дзен) с ссылкой на сайт.
   - Карточки в каталогах сервисов/SaaS (Startpack, Otzyvmarketing, профильные подборки «сервисы для селлеров»).

### 5.3. Типы доноров (от белых к серым)
| Тип | Примеры площадок | Ценность |
|---|---|---|
| Экспертные статьи / гостевые | VC.ru, Habr (аккуратно), Spark.ru, Cossa; e-com СМИ: Retail.ru, Oborot.ru, E-pepper.ru, New-Retail.ru | Высокий траст + трафик |
| Каталоги/рейтинги сервисов | Startpack, Otzyvmarketing, подборки «лучшие сервисы аналитики WB» | Тематичность, реф-трафик |
| Отзовики/обзоры | Otzovik, iRecommend, профильные обзоры | Бренд-сигналы, доверие |
| Контентные площадки | Дзен, Pikabu (нативно), VC-комментарии | Охват + разбавление профиля |
| **Крауд-маркетинг** | тематические форумы, Telegram/VK-чаты селлеров (ссылка в контексте, с UTM) | Дёшево, реф-трафик, естественность |
| Партнёрские ссылки | страницы «Партнёры/Интеграции» у сервисов-партнёров | Тематичные dofollow |
| **Биржи (закупка)** | GoGetLinks, Miralinks (вечные статейные), Kwork (гест под ключ); для Google — зарубежный аутрич | Масштаб, но осторожно |

> ❌ **Не использовать на молодом домене:** массовую аренду дешёвых ссылок (Sape-аренда), прогоны, PBN, пачки однотипных анкоров — прямой путь под **Минусинск** (Яндекс) / **Penguin** (Google).

### 5.4. Анкор-стратегия (естественное распределение)
- **Брендовые** (`Veloseller`, `veloseller.ru`): **40–55%**
- **Безанкорные / URL / «тут», «сайт», «здесь»**: **20–30%**
- **Разбавочные общие** («сервис аналитики для маркетплейсов», «аналитика остатков»): **15–20%**
- **Точные коммерч. ключи** («аналитика остатков Wildberries» и т.п.): **< 10%** и только на качественных донорах.
- Главное — **разнообразие и неравномерность**, без переоптимизации.

### 5.5. Тиринг
- **Tier-1** (качественные: СМИ, каталоги, гест) → на **страницы-акцепторы**: главная, лендинги модулей (Остатки / Неликвид / Radar), опорные статьи блога.
- **Tier-2** (крауд, соцсигналы, закладки) → **усиление Tier-1** и ускорение индексации.

### 5.6. График запуска ссылок (нельзя резко!)
| Период | Темп | Фокус |
|---|---|---|
| **Мес 0** (подготовка) | 10–20 «брендовых»/крауд естественно | Справочники, каталоги, соцсети, аудит |
| **Мес 1** | 5–10 / мес | 2–3 статейных/гест + каталоги + крауд |
| **Мес 2–3** | 8–15 / мес | + гостевые в e-com СМИ, первые кейсы |
| **Мес 4–6** | 10–20 / мес | Тематические СМИ, кейсы, Tier-2 усиление |

Принцип: **плавный, неравномерный рост**, синхронно с выходом контента и ростом брендовых запросов. Резкий скачок ссылочного на молодой сайт = риск фильтра.

### 5.7. Бюджет на ссылки (ориентиры РФ, проверить при запуске)
- Старт можно сделать **почти бесплатно**: крауд + каталоги + аутрич за экспертизу/бартер.
- Вечная статейная на приличном доноре (GoGetLinks/Miralinks): **~1 500–6 000 ₽**.
- Гостевая в тематическом СМИ: **от 5 000–30 000+ ₽**.
- **Реалистичный старт: 30 000–80 000 ₽/мес**, масштабировать строго по результату (реф-трафик + позиции).

### 5.8. Контроль и метрики линкбилдинга
- **Реестр доноров** (Google Sheets): площадка · тип · анкор · URL-акцептор · дата · цена · проиндексирована? · реф-трафик · статус.
- Динамика: реф-домены, ИКС (Яндекс), DR/UR (ahrefs), доля проиндексированных ссылок.
- Рост **позиций по кластерам** (Я.Вебмастер / Топвизор / GSC).
- **Реф-трафик** по UTM с каждого донора.
- **Токсичность**: мусорные ссылки → Disavow (Google) / чистка (Яндекс).

## 6. Контент-план (SEO-фундамент под ссылки)

Линкбилдинг без контента бессмыслен — сначала страницы, на которые не стыдно ставить ссылку.

**Кластеры ключей (семантика):**
- «аналитика для Wildberries / Ozon», «сервис аналитики маркетплейсов»
- «учёт остатков на WB/Ozon», «как не уйти в out-of-stock»
- «неликвид / замороженные деньги / оборачиваемость склада»
- «FBO vs FBS — отличия», «юнит-экономика WB/Ozon», «комиссия WB»
- «что продавать на WB — поиск ниши» (под Radar/Wordstat), «потерянная выручка»

**Форматы (что хорошо линкуется):**
- **Гайды** и разборы (экспертные, длинные).
- **Лид-магниты / linkbait:** бесплатный **калькулятор потерянной выручки**, калькулятор оборачиваемости, шаблон Excel-учёта, чек-лист «как не уйти в стокаут». Это собирает ссылки **естественно** и приводит trial.
- **Кейсы клиентов** с цифрами.
- **Сравнения**: «Veloseller vs Excel», «обзор сервисов аналитики для WB».
- **Глоссарий** терминов (стокаут, FBO/FBS, неликвид…) — ловит низкочастотку.

## 7. Дорожная карта на 90 дней

**Дни 1–30 — фундамент**
- Я.Вебмастер + GSC + Метрика/GA, UTM-разметка, цели/воронка.
- Семантика + 6–8 опорных статей + **1 лид-магнит** (калькулятор).
- Базовые регистрации: справочники, каталоги SaaS, соцсети бренда.
- Зайти в 10–15 сообществ селлеров, начать давать пользу.
- Запустить реферальную программу.

**Дни 31–60 — конвейер + старт ссылок**
- 2–4 материала/нед; старт линкбилдинга (крауд + 1–2 гостевых).
- 2–3 интеграции с инфлюенсерами (бартер).
- Тест Яндекс.Директа на горячие ключи (маленький бюджет).

**Дни 61–90 — масштабирование**
- Усиливаем каналы, давшие лучшую unit-экономику.
- Гостевые в e-com СМИ, публикация кейсов.
- Расширение закупки ссылок по результату; A/B лендингов.

## 11. Антипаттерны (чего не делать)
- Не лить пачку дешёвых/арендных ссылок на молодой домен (Минусинск).
- Не переспамливать точные анкоры — следить за распределением (§5.4).
- Не спамить в чатах — забанят и сожжём репутацию; только польза.
- Не накручивать отзывы грубо — модерация площадок + риск доверия.
- Не запускать платный трафик до того, как лендинг конвертит в trial.
